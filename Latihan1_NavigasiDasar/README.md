# 📱 Compose Multiplatform Navigation App

Aplikasi sederhana menggunakan **Kotlin Compose Multiplatform** dengan 2 layar:
- Home Screen
- Detail Screen

---

## ✨ Fitur
- Navigasi antar screen (Navigation Compose)
- UI modern (Material 3)
- Desain biru dengan gradient
- Struktur project rapi (navigation & screens)

---

## 🔄 Navigasi
- Start: `home`
- Home → Detail: `navigate("detail")`
- Detail → Home: `popBackStack()`

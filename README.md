# Tugas Praktikum 5 - Navigasi Android

Repositori ini berisi kumpulan latihan dan tugas praktikum mengenai implementasi sistem navigasi di Android. Proyek ini mencakup dasar-dasar perpindahan halaman hingga penggunaan navigasi yang lebih kompleks.

## 📂 Daftar Isi
* **Latihan 1: Navigasi Dasar** – Implementasi perpindahan antar layar (Activity/Fragment/Screen).
* **Latihan 2: Navigasi Dengan Argument** – Mengirimkan data antar layar menggunakan argumen.
* **Latihan 3: Bottom Navigation** – Implementasi menu navigasi di bagian bawah layar.
* **Tugas Praktikum 5** – Proyek utama yang menggabungkan seluruh konsep navigasi yang telah dipelajari.

---

## 🛠️ Teknologi yang Digunakan
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose / XML (Sesuaikan dengan yang kamu pakai)
* **Navigation Component:** Jetpack Navigation
* **IDE:** Android Studio

---

## 🚀 Cara Menjalankan Proyek
1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/Gianlvander/5_123140040.git
   ```
2. Buka **Android Studio**.
3. Pilih **Open an Existing Project** dan arahkan ke folder yang sudah di-clone.
4. Tunggu proses **Gradle Sync** selesai.
5. Jalankan aplikasi di **Emulator** atau **Physical Device**.
